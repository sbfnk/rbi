# Update a libbi object

This updates all the time stamps in a libbi object; it is useful after
(input, output, etc.) files have been changed outside the object itself.

## Usage

``` r
# S3 method for class 'libbi'
update(x, ...)
```

## Arguments

- x:

  a [`libbi`](https://sbfnk.github.io/rbi/reference/libbi.md) object

- ...:

  ignored

## Value

a [`libbi`](https://sbfnk.github.io/rbi/reference/libbi.md) object with
updated timestamps
