# Get the LibBi version

This tries to find the libbi executable to determine the installed
version of LibBi

## Usage

``` r
installed_libbi_version(...)
```

## Arguments

- ...:

  any arguments for
  [`locate_libbi`](https://sbfnk.github.io/rbi/reference/locate_libbi.md)
  (esp. `path_to_libbi`)

## Value

a character string with the installed version
