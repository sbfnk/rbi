# Check if a model is empty

Checks if a model is empty (i.e., has been initialised without any
content)

## Usage

``` r
is_empty(x)
```

## Arguments

- x:

  a [`bi_model`](https://sbfnk.github.io/rbi/reference/bi_model.md)
  object

## Value

TRUE or FALSE, depending on whether the model is empty
