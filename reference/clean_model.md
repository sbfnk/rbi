# Strip model code to its bare bones

Cleans the model by working out correct indents, removing long comments
and merging lines

## Usage

``` r
clean_model(x)
```

## Arguments

- x:

  a [`bi_model`](https://sbfnk.github.io/rbi/reference/bi_model.md)
  object

## Value

the updated `bi_model` object
