# Internal function to create a temporary working folder

The folder will be removed when the object is destroyed

## Usage

``` r
create_working_folder(x)
```

## Arguments

- x:

  a [`libbi`](https://sbfnk.github.io/rbi/reference/libbi.md) object

## Value

a [`libbi`](https://sbfnk.github.io/rbi/reference/libbi.md) object with
updated working folder
